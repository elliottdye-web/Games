# Games

A collection of games built with Python and Pygame.

## Projects

### Snake Game
A classic Snake game where you control a snake to eat food and grow longer. Avoid hitting the walls and yourself!

**How to Play:**
- Use arrow keys to control the snake direction
- Eat the food (red squares) to grow
- Don't hit the walls or yourself
- The game ends when you crash

**Requirements:**
- Python 3.7+
- Pygame

**Installation:**
```bash
pip install pygame
```

**Running the Game:**
```bash
python snake_game.py
```

## Controls
- **Arrow Keys** - Move the snake up, down, left, right
- **SPACE** - Pause/Resume the game
- **Q** - Quit the game

Enjoy playing!
